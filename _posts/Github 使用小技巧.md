### 高级搜索

- 按 *s* 定位到输入框
- 输入一串代码 如：spring vue

	-  react star:> 10000 pushed:> 2022-05-02 language: TypeScript
- 点击首页搜索左下角的 *Cheat sheet* 文案 即可查看高级搜索配置功能 [搜索 cheat sheet 备忘录表](https://github.com/search?q=search+cheat+sheet)
